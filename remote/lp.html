Linear Processing Travel Booking System

Background

My team of 3 were tasked with creating a proof of concept travel booking system using Natural User Interface gestures, built in WPF and Microsoft Expression Blend.

Process

With an understanding of the client's functional requirements, we ran a brainstorming session with the team devising possible activity flows and mocked up interface components.

Having established a near-complete end-to-end process, I then translated this into wireframes and annotations as a deliverable to the client, programmers and visual designer.

This project required a degree of fluidity that could accommodate rapid design changes. It also needed to showcase interactivity and animation - which meant traditional wireframe assets were not suitable in this situation.

Using the completed visual design, I used those elements to construct a high-fidelity prototype developed in PowerPoint. Equipped with a touch screen monitor, PowerPoint allowed for rapid prototyping while testing transitions, timings and feel. The client was more immersive in providing suggestions using this technique.


  • Client: Linear Processing
  • Project: Travel booking system
  • Date: March 2010 - September 2010
  • Service: Interaction design, WPF/Silverlight
