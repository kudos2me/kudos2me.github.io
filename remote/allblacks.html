Official All Blacks Windows Phone 7 App

I was tasked with creating an interface for the official All Blacks app for the Windows Phone 7 mobile. The app retrieved news, fixtures and player content from an external XML data source.

After sketching conceptual wireframes, I quickly developed lo-fi prototypes reflecting the app's UI and transitions between each card using the Expression Blend tool.

An advantage to prototyping Windows Phone 7 Apps is that the prototyping tool is baked into the same development environment used to create the final version. I used this project as an exercise to learn more about Sketchflow and Expression Blend.

I worked closelye with our visual designer to provide direction on the capabilities with Windows Phone 7 platform.

With visual design approved, elements of the prototype's code base were easily translated into the final build.

I was also responsible for integrating the visuals, designed in Photoshop, with the backend code developed by our programmers.

I leverage this technical "know-how" of Silverlight by creating interfaces that meet user needs, business goals and are realistic to develop.
