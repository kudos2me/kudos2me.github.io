Camden Data - Camden Plan

The Camden Plan is a microsite designed to inform Camden residents about it plans for future growth within the borough. The site had to be content editable and delivered within 3 days on a SharePoint 2010 platform.

Main deliverables included:

  • Custom page layouts
  • Enabling content owners to add content using Rich HTML content user controls
  • Translating PhotoShop files into working HTML templates
  • Embedding Google Analytics
  • Creating instructional guide for end-users

Visual compositions were supplied by a third-party design agency. I provided guidance on what was achieveable on the SharePoint platform within the timeframe.

I then created a custom aspx page layout with Rich HTML user controls. All design elements referenced a seperate stylesheet specific to this microsite for modularity. Tables were created using out-of-the-box content editors with custom styles created using the SharePoint method.
