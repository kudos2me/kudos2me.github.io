New Zealand Customs Service Public Web

For the NZ Customs public website, I was the lead front end developer on this SharePoint 2010 build and was responsible for developing the master page, internal pages, integrating jQuery controls and setting up the css framework.

Being a Government website, the site had to adhere to the New Zealand Web Standards, which is based on the international Web Content Accessibility Guidelines 2.0. This meant that it had to be accessible to all users on most standard web browsers. Developing towards this on the SharePoint platform proved one of the real challenges, which we successfully implemented.

  • Project: Public website
  • Date: October 2010 - January 2011
